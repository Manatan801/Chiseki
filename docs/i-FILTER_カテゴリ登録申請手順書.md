# i-FILTER URLカテゴリ登録申請 手順書

## 概要

職場環境で `https://gomi-maru.com/`（ゴミ丸スラッシュ地籍）にアクセスした際、
**i-FILTER（デジタルアーツ社）** により「推奨フィルター（カテゴリ外）」としてブロックされています。

これは、サイトが i-FILTER のURLカテゴリデータベースに未登録であることが原因です。
以下の手順で登録申請を行うことで、ブロックを解除できます。

---

## ブロック情報（確認済み）

| 項目 | 内容 |
|------|------|
| URL | `https://gomi-maru.com/` |
| ブロック理由 | 推奨フィルター（カテゴリ外） |
| IPアドレス | 99.99.99.32 |
| ブロック製品 | i-FILTER（Digital Arts Inc.） |
| 確認日時 | 2026/04/06 17:19:40 |

---

## 対処方法

### 方法1: ホワイトリスト追加（即時対応）

職場のIT管理者（情報システム担当）に以下を依頼してください。

**依頼内容：**

> i-FILTERの管理画面で、以下のURLをホワイトリスト（許可リスト）に追加してください。
>
> - 対象URL: `gomi-maru.com`（ワイルドカードで `*.gomi-maru.com` も含む）
> - 理由: 業務で利用する自治体向け地籍調査支援サービスのサイトです

**特徴：**
- 即時反映される
- ただし、自社（自組織）環境のみの対応

---

### 方法2: デジタルアーツへのカテゴリ登録申請（恒久対応・推奨）

i-FILTERのURLデータベースに正式登録されれば、全ての i-FILTER 導入組織でアクセス可能になります。

#### 申請フォーム

**URL:** `https://forms.daj.co.jp/bs/request/if_url/`

※ デジタルアーツ公式サイト（https://www.daj.jp/support/contact/）からもアクセス可能

#### 申請に必要な情報

| 項目 | 必須 | 記入内容 |
|------|:----:|----------|
| ご利用中の製品・サービス | ○ | 「i-FILTER」を選択（※環境に応じて i-FILTER@Cloud 等を選択） |
| バージョン | ○ | 利用中のバージョンを選択 |
| シリアルNo | ○ | i-FILTERのライセンスシリアル番号 |
| 組織名 | ○ | 所属組織名 |
| 所属部署 | - | 任意 |
| ご芳名 / フリガナ | - | 任意 |
| 対象サイトURL | ○ | `https://gomi-maru.com/` |
| 登録理由 | ○ | **「登録漏れ」** を選択 |
| ご意見・ご要望 | - | 下記の記載例を参照 |

#### ご意見・ご要望欄の記載例

```
当該サイトは自治体向け地籍調査支援サービスを提供するWebサイトです。
業務利用のため、適切なカテゴリ（行政/公共サービス等）への登録をお願いいたします。
```

#### 申請手順

1. 上記フォームURLにアクセス
2. 各項目を入力（シリアルNoはIT管理者に確認）
3. 個人情報の取り扱いに同意
4. 送信

#### 処理期間

- **通常1営業日程度** でカテゴライズされます
- データベースは1日3〜4回更新されます
- 申請結果の個別連絡はありません（反映後にアクセスして確認）

---

### 方法2-B: i-FILTER管理画面からの申請（Ver.10.40R01以降）

IT管理者が i-FILTER の管理画面から直接申請することも可能です。

1. i-FILTER管理画面 →「システム」→「URLカテゴリ判定」
2. `gomi-maru.com` のカテゴリ判定を実行
3. 「カテゴリ変更申請」をクリック
4. 希望カテゴリを選択（不明な場合は「選択しない」でも可）
5. 「申請する」をクリック
