# DXF作図条件書（自動読み取り用）

## 概要

本ツール（地籍調査DX帳票自動生成）は、DXFファイルから **CIRCLE（杭位置）**、**TEXT/MTEXT（杭番号・地番等）**、**LINE/LWPOLYLINE（境界線）** の3要素を読み取り、グラフ構造を構築して結線指示票・交点計算指示書を自動生成します。

以下の条件を満たすDXFであれば正確に処理できます。

---

## 1. 杭（CIRCLE）

**すべての杭位置にCIRCLEを配置すること。**

- 現況図内のすべての境界点（杭・交点・既設杭・復元杭・計算杭）の位置に、CIRCLEエンティティを配置する。
- CIRCLEの中心座標が杭の正確な座標となる。
- 半径は図面内で統一する（複数の半径があっても読み取り可能だが、統一が望ましい）。
- CIRCLEのないLINE収束点は杭として認識されず、その点を通る境界線がグラフに反映されない。

---

## 2. 杭番号（TEXT / MTEXT）

**各CIRCLEの近くに杭番号のTEXT（またはMTEXT）を1つ配置すること。**

- 杭番号テキストの位置は、対応するCIRCLEの中心からなるべく近くに置く。
- 1つのCIRCLEに対して杭番号テキストは1つだけ。
- 番号なしのCIRCLEは帳票に出力できない。

### 対応する杭番号の記法

| 接頭辞 | 意味 | 例 |
|--------|------|-----|
| なし | 新設杭 | `14.235` |
| `既` | 既設杭 | `既14.001` |
| `交` | 交点杭 | `交16.473` |
| `復` | 復元杭 | `復15.466` |
| `計` | 計算杭 | `計16.5002` |
| `-`（ハイフン） | 交点杭（従来形式） | `-13.298` |

### ブロック番号

杭番号の整数部分がブロック番号として扱われる（例：`14.235` → ブロック14）。交点計算指示書はブロック番号ごとにシートが分割される。

---

## 3. 境界線（LINE / LWPOLYLINE）

**境界線の端点はCIRCLEの中心座標と完全に一致させること。**

- 各境界線は2点のLINE（またはLWPOLYLINE）とする。
- 両端の座標がCIRCLEの中心座標と **一致（誤差0.5単位以内）** であること。
- 端点がCIRCLE中心からずれていると、その線分は境界線として認識されない。

---

## 4. レイヤーの分離

**境界線と注釈線は必ず別レイヤーに分けること。**

これは最も重要な条件です。

- **境界線レイヤー**: 実際の筆界を示すLINEのみを配置する。
- **注釈レイヤー**: 参照線・引出線・寸法線・メモ用の線など、筆界ではないLINEは別レイヤーに分ける。
- **背景レイヤー**: 地形図・地図等の背景は専用レイヤーに分離する。

### 混在させてはいけない理由

1つのレイヤーに境界線と注釈線が混在すると、注釈線が偽の接続（ショートカット）を生成し、結線順序が誤る。

例：杭A→杭B→杭C→杭Dの境界が正しいとき、注釈線でA→Cが接続されると、杭Bが結線から脱落する。

---

## 5. 地番・地目（TEXT / MTEXT）

- 地番テキスト（例：`1573-3`）は、対応する筆の **内部** に配置する（筆の面判定に使用するため）。
- 地目テキスト（例：`宅地`、`山林`）は地番テキストの近くに配置する。

### 対応する地目

宅地、山林、雑種地、公衆用道路、田、畑、原野、墓地、境内地、学校用地、鉄道用地、池沼、牧場、保安林、ため池

### 対応する公共用地ラベル

`県道-5`、`河川-2`、`市道-1`、`国道-3`、`水-1`、`道-2` 等（種別＋番号の形式）

---

## 6. 対応するDXFバージョン・エンコーディング

| 項目 | 対応形式 |
|------|---------|
| DXFバージョン | AC1009（AutoCAD R12）以降すべて |
| テキストエンティティ | TEXT、MTEXT の両方 |
| 線分エンティティ | LINE、LWPOLYLINE の両方 |
| エンコーディング | Unicode、Shift-JIS の両方（自動判定） |

---

## チェックリスト

DXFファイルを本ツールで処理する前に、以下を確認してください。

- [ ] すべての境界点にCIRCLEがある
- [ ] すべてのCIRCLEの近くに杭番号テキストがある
- [ ] 境界線の端点がCIRCLE中心と一致している
- [ ] 境界線と注釈線が別レイヤーに分かれている
- [ ] 地番テキストが筆の内部に配置されている
- [ ] 背景図（地形図等）は専用レイヤーに分離されている
