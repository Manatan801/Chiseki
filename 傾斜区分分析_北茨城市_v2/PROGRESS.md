# 傾斜区分分析ツール v2 進行記録

更新日: 2026-06-18

## 実装内容

- 現行版 `傾斜区分分析_北茨城市` を残し、v2 を `傾斜区分分析_北茨城市_v2` として別フォルダ化。
- 傾斜計算を `np.gradient()` 方式から、DEM の 3x3 近傍 9 点による最小二乗平面フィット方式へ変更。
- 面積計算は地図ズームではなく、DEM の実範囲と配列サイズから求めた格子寸法で算出。
- v2 の起動ポートを `8768` に変更。
- 地図初期表示を最大ズーム `17` に変更。
- 高詳細地図タイル `z15` から `z17` を事前収録し、オフライン PC で利用できる構成に変更。
- UI、CSV、操作マニュアルへ v2 の計算方式と DEM 格子寸法を表示。

## 収録データ

- DEM: `data/kitaibaraki_dem.npz`
- 背景地図タイル: `data/map_tiles/`
- 収録ズーム: `z10` から `z17`
- 高詳細地図: `z15` から `z17` まで北茨城市 DEM 範囲分を収録済み

## 検証内容

- Python 構文チェック。
- 人工 DEM 平面で理論傾斜と一致することを確認。
- 平坦 DEM で傾斜 0 度になることを確認。
- NoData を含む 3x3 近傍が集計対象外になることを確認。
- 実 DEM の小ポリゴンで解析 API が正常応答することを確認。
- `z17` 地図タイルがローカル HTTP から PNG として返ることを確認。
- 解析 API の面積計算が DEM 格子寸法 `約3.823m x 約3.823m` のまま維持されることを確認。

## 注意

- v2 は完全オフライン PC への配布を想定。
- 面積・傾斜計算は背景地図の表示ズームに依存しない。
- 現行版フォルダは変更していない。
- `data/` と `wheels/` は大容量バイナリのため Git 管理外。オフライン PC へ実装するときは v2 フォルダ内の同梱データとして一緒にコピーする。
