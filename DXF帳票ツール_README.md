# DXF帳票ツール

JW-CAD等のDXF図面から、次の帳票を生成するツールです。

- `結線指示票.xls`
- `交点計算指示書.xlsx`

## Windowsへ移すファイル

配布フォルダごとWindowsへコピーしてください。
フォルダ内の構成は変えないでください。

```text
DXF帳票ツール_Windows配布/
  DXF帳票ツール起動.bat
  初回セットアップ.bat
  DXF帳票ツール_README.md
  dxf_report_web.py
  requirements.txt
  wheels/
  src/
  data/templates/
```

## 必要なもの

- Python 3.12 64bit
- インターネット接続は不要

## 初回だけ行うこと

`初回セットアップ.bat` をダブルクリックしてください。

`wheels` フォルダ内のファイルから、必要なPythonライブラリをオフラインでインストールします。

この配布フォルダに入っているwheelは `Windows 64bit + Python 3.12` 用です。
Python 3.10 / 3.11 / 3.13 ではインストールできない場合があります。

## 起動方法

`DXF帳票ツール起動.bat` をダブルクリックしてください。

ブラウザで次の画面が開きます。

```text
http://127.0.0.1:8766/
```

## 使い方

1. DXFファイルを選択します。
2. 作成する帳票を選択します。
3. 必要なら対象地番をカンマ区切りで入力します。
4. `帳票を生成` を押します。
5. 生成された帳票ZIPがダウンロードされます。

ZIPの中に `結線指示票.xls` と `交点計算指示書.xlsx` が入ります。

## 注意

生成された帳票は必ず人が確認してください。
複雑な交点や図面ルールから外れた箇所は、自動判定できない場合があります。
